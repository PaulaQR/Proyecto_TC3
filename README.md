### ¿ Cómo utilizar nuestra aplicación ? 

Vas a ingresar tú nombre de usuario,
Ingresas los datos de tú tarjeta al finalizar oprimes el botón de Validar y te saldrá 
si tú tarjeta es VALIDA o INVALIDA
